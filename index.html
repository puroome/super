👩‍🏫 감독교사 목록

CSV 6열: 이름, 이전누적업무강도, 배정불가고사실, 못들어가는시간, 반드시들어가야하는시간, 감독유형
여러 항목 구분: 세미콜론(;) 사용 (예: 101;201 / 1_3;2_1)
반드시들어가야하는시간과 감독유형은 개수가 반드시 일치해야 합니다. (1=정감독, 2=부감독)
고사실명에 "복도" 포함 시 자동으로 부감독(2)으로 처리됩니다.

📊 고사실별 배정감독수 설정

각 날짜/교시/보직별로 고사실에 몇 명을 배정할지 입력하세요.

CSV 형식: 날짜,교시,보직,고사실1,고사실2,... (첫 줄은 헤더)
날짜·보직명은 기본정보 탭에 등록된 값과 정확히 일치해야 합니다.
⬇ 버튼으로 현재 값이 채워진 양식을 받아 엑셀에서 수정 후 다시 가져오세요.

🎯 자동배정

셀 클릭: 선택(swap용)  |  셀 더블클릭: 고정 토글
⚙️ 배정 전 확인사항
· 교사별 "배정할시간" 합계 = 전체 필요감독 수 합계여야 합니다.
· 배정불가 고사실은 쉼표로 구분해 입력하세요 (예: 1-1,2-3)
· 고정셀(더블클릭)은 자동배정에서 변경되지 않습니다.

자동배정을 실행하거나 저장된 배정을 불러오세요.
