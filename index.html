📖 사용 설명서

⚙️ 기본정보 → 📝 배정설정 → 🎯 자동배정 → 🖨️ 감독표 순서대로 진행하세요.
앞 단계를 완료해야 다음 탭으로 진행가능합니다.
1 ⚙️ 기본정보
  • 시험일 및 고사실
    [+ 날짜 추가]로 시험 날짜와 시작/종료 교시 입력
    고사실 등록은 [고사실CSV 올리기]로만 가능
    CSV 열: 고사실명, 부감독(값이 있으면 부감독실, 비우면 정감독실)
  • 감독유형 및 업무강도
    정감독은 강도 100으로 고정이며, 부감독은 업무강도 직접 결정
    [부감독 비활성화]를 누르면 모두 정감독으로 배정
  • 감독교사 목록
    [+ 교사 추가] 또는 [CSV 올리기]로 명단 입력
    각 교사의 이전누적강도(지난 시험까지 쌓인 업무량)와 제외 고사실을 입력
    이전누적강도는 [📊 이전누적강도 불러오기]로 목록에서 선택 가능
    CSV 열: 이름, 이전누적강도, 제외고사실, 제외시간, 고정시간, 감독유형
        - 제외고사실 : 본인 담임반에 배정되면 안되는 등의 사유로 제외시킬 고사실
        - 제외시간 : 출장 등 사유로 감독을 할 수 없는 시간
        - 고정시간 & 감독유형 : 담당과목 등의 사유로 반드시 감독배정되어야 하는 시간 (감독유형도 함께 결정)     - 제외시간, 고정시간, 감독유형 값은 [자동배정]에서 직접 입력가능
2 📝 배정설정
    각 교시에 어떤 고사실을 운영할지 표에서 설정
    셀을 더블클릭해서 직접 배정(다시 더블클릭시 미배정) or [CSV 올리기]로 고사실별 운영여부 결정
    색 구분 : 초록 (정감독), 노랑 (부감독)
3 🎯 자동배정
    [🚀 자동배정 실행]을 누르면 감독이 자동 배정
    자동배정 실행전 필요시, 1단계의 [감독교사 목록]에서 누락한 제외시간, 고정시간, 감독유형 직접입력
        - [🚫 감독배정 제외] 버튼을 클릭 후, 셀을 선택하여 제외
        - [📌 감독시간 고정] 버튼을 클릭 후, 셀을 선택하여 시간고정 (이후 우클릭해서 감독유형 입력 필수)
        - 감독 교환 : 교환할 두 셀을 선택 후, [⇄ 선택 셀 교환] 버튼 클릭
        - 배정 취소 : 자동배정된 결과를 취소 (단, 감독배정 제외, 감독시간 고정은 남아있음)
        - 변경 취소 : 자동배정 후 실행한 모든 변경사항을 취소
        - 더블클릭 : 배정된 셀을 더블클릭시 배정결과(감독실+시간)을 고정해서, 자동배정 실행해도 안바뀜
4 🖨️ 감독표
    [🖨️ 전체 감독표] : 전체 배정표 인쇄 or 파일 저장
    [🖨️ 개인 (전체)] : 전체 교사의 개인별 감독표 인쇄
    드롭다운에서 교사 목록 중 원하는 개별 교사 선택해 인쇄
＋ 상단 메뉴 (저장·관리)
    저장 : 지금 작업본에 덮어쓰기
    다른 이름 저장 — 새 이름의 저장본 만들기
    불러오기 : 저장해 둔 목록에서 불러오기
    전체 초기화 : 현재 화면의 모든 데이터 삭제 (저장해 둔 목록 영향X)
